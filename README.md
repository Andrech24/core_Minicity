# MiniCity Core

Proyecto nuevo hecho con Spring Boot y Angular.

El objetivo es administrar el core de MiniCity con datos reales guardados en Neon PostgreSQL.

Incluye:

- Login simple.
- Dashboard del core.
- Listado de ninos desde Neon.
- Registro rapido de ninos.
- Edicion y eliminacion de ninos.
- Sistema de puntos por cada nino.
- Comparacion de temporada entre mes anterior y mes actual.
- Listado y registro de reservas.
- Tipos de entrada desde base de datos.
